命令相关

查看 nginx 状态: systemctl status nginx.service

重启 ssh: systemctl restart sshd

重启 nginx:systemctl restart nginx.service

pm2 查看进程：pm2 ls

pm2 启动进程： pm2 start <id> 第一次启动时需要到指定的启动脚本目录下执行pm2 start <js文件名称>

pm2 停止进程：pm2 stop <id>

Tips :

​ 上诉操作如果遇到权限问题的话可以尝试在命令行前面添加 sudo 解决
